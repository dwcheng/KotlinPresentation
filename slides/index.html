Kotlin

Why?

  • Java Interop
  • Modern Language Features
    • Concise Syntax
    • Null Safety
  • Interop eliminates big pain point, rewriting whole sections
  • Allows incremental rewrites, meaning you can migrate one file at a time
  • When I say concise, I mean very concise
  • One of the big complaints against Java is that it is wordy
  • Kotlin borrows heavily from other languages and is like a melting pot
  • Biggest advantage is that NPEs are now a compile time error
  • If your code could throw a NPE, it MUST be dealt with
